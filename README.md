## 介绍

FMCL (Functional Minecraft Launcher) 是一个用Python编写的基于PyQt5的跨平台的MC启动器

## 贡献

您可以做一下几件事或者更多:

- 添加更多功能
- 修复Bug
- 翻译
- 完善帮助

### 准备

在根目录执行以下命令

```shell
pip install -r requirements.txt
```
